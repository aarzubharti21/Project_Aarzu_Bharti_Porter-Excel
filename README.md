# 📌 Project Overview

This project analyzes Porter delivery operations data to identify key factors affecting delivery performance, operational efficiency, and customer experience.

The objective of this project is to transform raw delivery data into actionable business insights using Microsoft Excel. The analysis focuses on delivery performance, order management, partner utilization, market performance, and operational bottlenecks.

The project includes data cleaning, KPI development, pivot table analysis, interactive dashboard creation, and business recommendations.

---

# 🎯 Business Objectives

The project addresses the following business questions:

### Delivery Performance Analysis

* What is the average delivery time?
* Which markets experience the highest delays?
* How efficiently are deliveries completed?

### Operational Efficiency

* How do outstanding orders affect delivery performance?
* What is the impact of busy delivery partners?
* How effectively are delivery resources utilized?

### Market Performance

* Which markets process the highest order volumes?
* Which markets generate the highest revenue?

### Customer Experience

* What operational factors contribute to longer delivery times?
* How can delivery efficiency be improved?

| store_primary_category   | Store Category            |
